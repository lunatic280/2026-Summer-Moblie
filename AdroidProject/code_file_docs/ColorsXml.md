# colors.xml

원본 파일: `app/src/main/res/values/colors.xml`

## 한 줄 역할

Android XML 리소스에서 사용할 기본 색상 값을 정의한다.

## 쉽게 말하면

XML 테마나 옛 Android View 시스템에서 참조할 수 있는 색상표다. 현재 Compose 화면은 대부분 `MusicPlayerColors.kt`의 색상을 직접 사용한다.

## 주요 색상

- `purple_200`
- `purple_500`
- `purple_700`
- `teal_200`
- `teal_700`
- `black`
- `white`

## 이 파일이 중요한 이유

앱에 XML 기반 테마나 리소스가 남아 있을 때 색상 참조를 제공한다. Compose 전용 색상과는 역할이 조금 다르다.
