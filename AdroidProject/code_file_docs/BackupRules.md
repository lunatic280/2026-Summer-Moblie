# backup_rules.xml

원본 파일: `app/src/main/res/xml/backup_rules.xml`

## 한 줄 역할

Android 자동 백업에 포함하거나 제외할 데이터를 설정하는 파일이다.

## 쉽게 말하면

사용자가 새 기기로 이동하거나 앱을 복원할 때 어떤 앱 데이터를 백업할지 정하는 규칙 파일이다.

## 현재 상태

현재는 템플릿 상태에 가깝고, 실제 include/exclude 규칙은 주석 처리되어 있다.

## 이 파일이 중요한 이유

나중에 DataStore나 Room으로 즐겨찾기, 최근 재생 기록을 저장하게 되면 이 파일에서 백업 정책을 명확히 정할 수 있다.
