# MusicPlayerColors.kt

원본 파일: `app/src/main/java/com/job/androidprojet/ui/home/MusicPlayerColors.kt`

## 한 줄 역할

음악 플레이어 화면에서 쓰는 색상을 한곳에 정의한다.

## 쉽게 말하면

홈 화면, 카드, 텍스트, 강조 색상을 통일하기 위한 색상 팔레트다.

## 주요 색상

- `HomeBackground`: 전체 배경
- `HomeSurface`: 카드나 리스트 배경
- `HomeSurfaceElevated`: 조금 더 떠 보이는 카드 배경
- `HomeAccent`: 재생 버튼, 선택된 필터 등에 쓰는 초록 계열 강조색
- `HomeTextPrimary`: 주요 텍스트
- `HomeTextSecondary`: 보조 텍스트
- `HomeTextMuted`: 덜 중요한 텍스트

## 이 파일이 중요한 이유

색상을 여기서만 관리하면 전체 UI 분위기를 쉽게 유지하거나 바꿀 수 있다.
