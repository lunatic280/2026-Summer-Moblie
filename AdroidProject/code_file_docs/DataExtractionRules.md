# data_extraction_rules.xml

원본 파일: `app/src/main/res/xml/data_extraction_rules.xml`

## 한 줄 역할

Android 12 이상에서 클라우드 백업과 기기 간 전송 규칙을 정의한다.

## 쉽게 말하면

앱 데이터가 Google 백업이나 새 기기 이전 과정에서 어떻게 다뤄질지 정하는 설정 파일이다.

## 현재 상태

현재는 기본 템플릿 상태이며, 구체적인 include/exclude 규칙은 아직 작성되어 있지 않다.

## 이 파일이 중요한 이유

앱이 실제 사용자 데이터를 저장하게 되면 어떤 데이터가 백업되어야 하는지, 어떤 데이터는 제외해야 하는지 여기에서 관리할 수 있다.
