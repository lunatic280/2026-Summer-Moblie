# 코드 파일 설명 문서 모음

이 폴더는 프로젝트의 주요 코드/설정 파일을 파일별로 쉽게 설명한 문서 모음이다.

읽는 순서는 아래를 추천한다.

1. `MainActivity.md`
2. `SampleMusicCatalog.md`
3. `Music.md`
4. `MusicPlayerViewModel.md`
5. `HomeScreen.md`
6. `LocalPlaybackController.md`
7. `MusicPlaybackService.md`
8. UI 컴포넌트 문서들
9. 테스트와 Gradle 설정 문서들

오디오 파일인 `res/raw/*.wav`, launcher 이미지, Gradle wrapper 바이너리는 코드 설명 대상이 아니라서 개별 설명 문서에서는 제외했다.
