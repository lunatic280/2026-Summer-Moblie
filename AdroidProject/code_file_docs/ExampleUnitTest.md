# ExampleUnitTest.kt

원본 파일: `app/src/test/java/com/job/androidprojet/ExampleUnitTest.kt`

## 한 줄 역할

로컬 JVM 단위 테스트 예시 파일이다.

## 쉽게 말하면

Android 기기 없이 개발 컴퓨터에서 실행되는 아주 기본적인 테스트다. 현재는 `2 + 2 = 4`를 확인하는 템플릿 테스트다.

## 주요 테스트

- `addition_isCorrect()`: 단순 덧셈 결과 확인

## 이 파일이 중요한 이유

현재 앱 로직 테스트의 핵심은 아니지만, 로컬 단위 테스트 구조가 잡혀 있음을 보여준다.
