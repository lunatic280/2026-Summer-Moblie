# Type.kt

원본 파일: `app/src/main/java/com/job/androidprojet/ui/theme/Type.kt`

## 한 줄 역할

앱의 기본 글자 스타일을 정의한다.

## 쉽게 말하면

Material3에서 사용할 typography 설정 파일이다. 현재는 `bodyLarge` 스타일만 기본값에서 조금 명시해둔 상태다.

## 주요 설정

- 기본 폰트 패밀리
- 기본 글자 굵기
- 글자 크기
- 줄 높이
- 글자 간격

## 이 파일이 중요한 이유

나중에 앱 전체 글자 스타일을 통일하고 싶을 때 이 파일에서 Typography를 확장하면 된다.
