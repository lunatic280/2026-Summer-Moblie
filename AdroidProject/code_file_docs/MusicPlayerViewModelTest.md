# MusicPlayerViewModelTest.kt

원본 파일: `app/src/test/java/com/job/androidprojet/ui/home/MusicPlayerViewModelTest.kt`

## 한 줄 역할

`MusicPlayerViewModel`의 상태 관리 로직을 검증한다.

## 쉽게 말하면

검색, 즐겨찾기, 다음 곡, 최근 선택 목록 같은 앱의 핵심 로직이 예상대로 동작하는지 확인하는 테스트 파일이다.

## 주요 테스트

- 검색어가 제목, 가수, 앨범 기준으로 필터링되는지 확인
- 곡 선택 시 현재 곡과 재생 상태가 바뀌는지 확인
- 즐겨찾기 토글이 Favorites 필터에 반영되는지 확인
- 다음 곡이 목록 끝에서 처음으로 돌아가는지 확인
- 최근 선택 목록이 중복 없이 최신순으로 유지되는지 확인
- 최근 선택 목록이 최대 8개로 제한되는지 확인
- 진행률이 0~1 범위로 보정되는지 확인
- 빈 목록에서는 재생 토글이 동작하지 않는지 확인

## 이 파일이 중요한 이유

실제 화면을 띄우지 않아도 ViewModel 로직이 맞는지 빠르게 검증할 수 있다.
