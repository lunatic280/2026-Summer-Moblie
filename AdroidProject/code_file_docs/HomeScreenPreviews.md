# HomeScreenPreviews.kt

원본 파일: `app/src/main/java/com/job/androidprojet/ui/home/HomeScreenPreviews.kt`

## 한 줄 역할

Android Studio 미리보기에서 여러 화면 상태를 확인할 수 있게 한다.

## 쉽게 말하면

앱을 실행하지 않아도 Compose Preview로 홈, 검색, 보관함, 플레이어, 미니 플레이어 등을 미리 볼 수 있게 만든 파일이다.

## 포함된 Preview 예시

- 기본 홈 화면
- 작은 화면 홈
- 검색 화면
- 검색 결과 없음
- 보관함 화면
- 플레이어 화면
- 긴 텍스트가 있는 플레이어
- 미니 플레이어
- 음악 목록이 비어 있는 상태

## 이 파일이 중요한 이유

화면이 잘리는지, 긴 제목이 깨지는지, 작은 화면에서도 보기 좋은지 빠르게 확인할 수 있다.
