# Color.kt

원본 파일: `app/src/main/java/com/job/androidprojet/ui/theme/Color.kt`

## 한 줄 역할

기본 Material3 테마 색상을 정의한다.

## 쉽게 말하면

Android Studio 템플릿에서 시작할 때 만들어지는 기본 색상 파일이다. 앱 전체 MaterialTheme의 primary, secondary, tertiary 색상에 쓰인다.

## 주요 값

- `Purple80`
- `PurpleGrey80`
- `Pink80`
- `Purple40`
- `PurpleGrey40`
- `Pink40`

## 이 파일과 MusicPlayerColors의 차이

`Color.kt`는 앱 전체 Material 테마용 기본 색상이고, `MusicPlayerColors.kt`는 음악 플레이어 화면에서 직접 쓰는 전용 색상이다.
