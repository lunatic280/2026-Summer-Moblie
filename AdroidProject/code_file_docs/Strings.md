# strings.xml

원본 파일: `app/src/main/res/values/strings.xml`

## 한 줄 역할

앱에서 사용하는 문자열 리소스를 정의한다.

## 쉽게 말하면

화면이나 Android 시스템 설정에서 쓰는 텍스트를 코드 밖에 모아둔 파일이다.

## 현재 문자열

- `app_name`: 앱 이름
- `playback_notification_channel_name`: 음악 재생 알림 채널 이름

## 이 파일이 중요한 이유

Android에서는 UI 텍스트나 알림 채널 이름을 리소스로 분리해두는 방식이 일반적이다. 다국어 지원을 할 때도 이 구조가 필요하다.
