# JulyFirstApplication

**작성일:** 2026-07-01

이 프로젝트는 Android 기본 UI 실습을 모듈별로 나눠 학습한 내용을 정리한 저장소입니다.  
각 모듈은 하나의 주제를 집중적으로 다루도록 구성되어 있습니다.

## 모듈별 학습 내용

| Module | 학습 내용 |
| --- | --- |
| `app` | 기본 화면 구성, `LinearLayout` 배치, `ImageView` 삽입, Edge-to-Edge와 시스템 바 패딩 처리 |
| `xmllayout` | XML 기반 레이아웃 구성, 세로/가로 `LinearLayout` 중첩, `layout_weight` 활용 |
| `viewattribute` | View 속성 적용과 기본 UI 구조 확인, Edge-to-Edge 처리 |
| `visibleclick` | 버튼 클릭으로 `VISIBLE` / `INVISIBLE` / `GONE` 상태를 변경하는 View 가시성 제어 |
| `test` | 체크박스, 라디오버튼, 버튼을 이용한 간단한 선택형 UI 구성 |
| `viewbindingexample` | View Binding 사용법, `binding` 객체로 View 접근, 버튼 클릭 시 텍스트 변경 |

## 정리

- XML 레이아웃의 기본 구조를 익혔습니다.
- 버튼, 텍스트, 이미지, 체크박스, 라디오버튼 등 기본 위젯을 사용했습니다.
- View Binding과 View 속성 변경을 통해 화면을 동적으로 제어하는 방법을 학습했습니다.
